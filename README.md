# مشروع سينمانا PS4 المتكامل (Final Build)

هذا هو الهيكل الكامل والنهائي لتطبيق سينمانا لجهاز PS4 المعدل، تم إعداده ليكون جاهزاً للبناء المباشر.

## هيكل المجلدات (أين تضع ملفاتك؟)

1. **`sce_sys/`**:
   - `param.sfo`: ملف معلومات التطبيق (يتم توليده تلقائياً أو استبداله بملفك الخاص).
   - `icon0.png`: أيقونة التطبيق (512x512). **[ضع صورتك هنا]**
   - `pic0.png`: صورة الخلفية عند اختيار التطبيق (1920x1080). **[ضع صورتك هنا]**
   - `pic1.png`: صورة إضافية تظهر في الخلفية. **[ضع صورتك هنا]**

2. **`sce_module/`**:
   - يحتوي على ملفات الـ `.prx` الضرورية لتشغيل التطبيق والمكتبات (مثل `libSceWebBrowser.prx`).
   - تم وضع ملفات فارغة كأماكن محجوزة، سيقوم الـ Toolchain باستبدالها بالنسخ الحقيقية أثناء البناء.

3. **`src/`**:
   - يحتوي على الكود المصدري `main.cpp` المحدث بحقوق **حيدر احمد - فريق 4GAMER**.

4. **`pkg.gp4`**:
   - ملف المشروع الرئيسي الذي يجمع كل ما سبق لإنتاج ملف الـ PKG النهائي.

## حقوق التطوير
- **المطور الرئيسي**: حيدر احمد
- **الفريق**: 4GAMER
- **الموقع المستهدف**: cinamana.shabakaty.com

## كيفية البناء
ارفع هذا المجلد بالكامل إلى GitHub. سيقوم الـ Workflow المرفق ببناء ملف `cinamana_final.pkg` تلقائياً وستجده في قسم الـ Actions.
